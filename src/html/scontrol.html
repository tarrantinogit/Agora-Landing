Security Control

All processes of implementation of functional modules of the platform are carried out in strict accordance with the best world practices used in Softengi. This is evidenced by the certification of Softengi according to the international standard ISO9001 and the international standard of information security management ISO27001.

Software development uses secure development practices based on the ASVS standard (OWASP project). All software components are penetration tested based on the OWASP Testing Guide.
